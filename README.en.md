# 老年人综合服务网站

#### 介绍

这是一个老小孩服务网站，本人大一时参加星火杯所做。内容包括登录页，首页，志愿队介绍页，个人健康页，推送页。

#### 软件架构

本网站使用css，JavaScript，html语言完成，在推送页设计了滑轮视差，在志愿队介绍页设计了动态页面。

#### 使用说明

1. 下载下来之后使用vscode即可运行
2. 在首页的邮箱一打开即可显示我的qq号以及面试问题
3. 健康页可以输入数据进行判断今日的身体情况
